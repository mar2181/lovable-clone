3. Add `NEXT_PUBLIC_CLERK_SIGN_UP_URL` = `/sign-up` for all environments (while you're there).
4. Trigger a Vercel redeploy (or a new deploy from master).
5. Verify: `curl -I https://hswebappbuilder.space/dashboard` → should return `307 Temporary Redirect` to `/sign-in`.

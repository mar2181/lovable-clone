4. **Fix planner over-scoping:** A single-sentence landing-page prompt should produce 1 page, not 4. Add a cap (`max_pages=1` for single-sentence prompts containing "landing page").

### P2 тАФ CORS `FALLBACK_ORIGINS` includes localhost in production [NEW]

`worker/src/index.ts` defines `FALLBACK_ORIGINS = ["http://localhost:3015", "http://127.0.0.1:3015", "https://hswebappbuilder.space"]`. The localhost entries are always merged into the production CORS allowlist. Browser CORS enforcement means this only matters if a browser tab is served from `localhost:3015` targeting the prod API, but it unnecessarily widens the allowed surface.

**Fix (repo change тАФ flag):**
```ts
// Move localhost entries out of FALLBACK_ORIGINS:
const FALLBACK_ORIGINS = ["https://hswebappbuilder.space"];
// Add to worker/.dev.vars:
// ALLOWED_ORIGINS=http://localhost:3015,http://127.0.0.1:3015
```
